После нажатия кнопки "Решить (II часть)" на экране будет отображена симплекс-таблица. 

Программа поддерживает 2 режима решения задачи симплекс методом:

1)                 пошаговый, с возможностью выбора опорного элемента;

2)                 автоматический.

Данные режимы совмещены. Для решения задачи в пошаговом режиме достаточно выбрать любой из возможных опорных элементов. Для решения задачи автоматически нужно нажать "Рассчитать автоматически", тогда программа сама продолжит поиск решения, начиная с отображаемой в данный момент симплекс-таблицы.

С помощью кнопок навигации, расположенных под таблицей, можно просмотреть предыдущие этапы решения.

Примечание: Начиная с симплекс-таблицы №1, в каждой из таблиц серым цветом выделено место расположения опорного элемента, выбранного ранее.

После некоторого числа шагов будет получен ответ задачи. Возможные варианты:

·        Будет найдено решение

·        Будет получено сообщение о неограниченности функции
