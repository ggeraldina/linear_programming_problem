Для решения задачи методом искусственного базиса в меню следует выбрать Файл/Метод/Решить с помощью метода искусственного базиса.  

Далее нужно ввести задачу. Это можно сделать 2 способами:

1)                   загрузить ранее сохраненную задачу (Для этого нужно в меню выбрать Файл/Открыть);

2)                   ввод задачи вручную

2.1)       Необходимо указать количество используемых переменных. Для этого нужно выбрать соответствующее число в выпадающем списке, расположенном рядом с фразой "число переменных".

2.2)       После следует сообщить программе количество ограничений задачи, т.е. выбрать соответствующее число рядом с фразой "количество условий". Количество условий не может быть больше числа переменных.

2.3)       Затем нужно выбрать в выпадающем списке, что следует искать минимум min или максимум max, и ввести коэффициенты целевой функции и коэффициенты ограничений задачи. О принимаемых программой форматах чисел (коэффициентов) можно узнать в соответствующем пункте справки.

 

Примечание: Важно ввести задачу в указанной ранее последовательности, так как максимально возможное количество условий зависит от числа переменных. А количество строк для ввода ограничений задачи зависит от введенного количества условий.

 

Решение задачи с помощью метода искусственного базиса разделено на 2 части:

I Часть   =>   поиск базиса с помощью метода искусственного базиса;

II Часть   =>   решение задачи симплекс методом.

 

После ввода задачи следует перейти к решению I Части. Для этого нужно нажать кнопку "Решить (I часть)". Если проверка введенных условий будет пройдена успешно, то тогда на экране будет отображена симплекс-таблица, в противном случае будет выведено сообщение об ошибке. 

Программа поддерживает 2 режима решения задачи:

1)                 пошаговый, с возможностью выбора опорного элемента;

2)                 автоматический.

Данные режимы совмещены. Для решения задачи в пошаговом режиме достаточно выбрать любой из возможных опорных элементов. Для решения задачи автоматически нужно нажать "Рассчитать автоматически", тогда программа сама продолжит поиск базиса, начиная с отображаемой в данный момент симплекс-таблицы.

С помощью кнопок навигации, расположенных под таблицей, можно просмотреть предыдущие этапы решения.

Примечание: В методе искусственного базиса представлены 2 типа чередующихся друг за другом таблиц: таблица с возможностью выбора опорного элемента и "промежуточная" таблица, полученная сразу после шага симплекс метода. В промежуточной таблице место расположения выбранного ранее опорного элемента выделено серым цветом; в следующей таблице (таблице с возможностью выбора опорного элемента) столбец с ранее выбранным опорным элементом будет отсутствовать, так как не представляет никакого интереса при дальнейшем расчете.

После некоторого числа шагов в итоге будет один из вариантов:

·        Будет найден базис

·        Будет получено сообщение о несовместности системы

·        Будет получено сообщение о зацикливании

В случае второго и третьего вариантов следует попробовать пересчитать таблицу, выбирая иные опорные элементы.

Если найден базис, тогда становится доступным переход к II Части решения. Для продолжения решения задачи симплекс методом нужно нажать кнопку  "Решить (II часть)". Подробнее о симплекс методе можно узнать из соответствующего пункта справки.
